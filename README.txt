This is a full-stack project using django-rest-framework(backend) and vue.js(frontend).

DESCRIPTION:
User admin of an insurance company can acces "/admin" endpoint in order to create custom templates for risk types. 

ENDPOINTS:
'/admin' -> User can create, modify and delete templates for risks 
'/api/risks/ -> List of all risks
'/api/risk/<ID> ->Retrieve single risk
'/risks' -> vue interface
